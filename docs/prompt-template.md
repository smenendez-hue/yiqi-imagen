# Plantilla de Prompt Funcional para Agentes

Copiar y completar:

```text
Contexto:
- Proyecto: <nombre>
- Objetivo funcional: <descripcion>
- Restricciones: <tecnicas/negocio>

Instrucciones:
1) Implementar la solucion completa.
2) Consultar y aplicar los lineamientos generales de YiQi Agent Standards (version vigente).
3) Usar Jest para casos de test del cambio.
4) Alinear UI a lineamientos de diseno YiQi.
5) Si hay integracion API, respetar contratos OpenAPI YiQi y manejo robusto de errores.
6) Validar build y tests al finalizar.

Entregables esperados:
- Archivos modificados
- Resumen de decisiones
- Resultado de build/tests
```
