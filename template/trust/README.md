# Trust stats template

Use this template for compact credibility metrics in heroes, banners, or product
sections.

![Trust cards preview](./assets/trust-cards-preview.png)

## Files

| File | Purpose |
|------|---------|
| `html/trust-inline.html` | Inline trust metrics with dividers. |
| `html/trust-cards.html` | Card trust metrics. |
| `html/trust-grid.html` | Dense grid trust metrics. |

## Style dependency

Load `https://diguardia.github.io/yiqi-imagen/styles.css`. Do not copy the full
stylesheet.
